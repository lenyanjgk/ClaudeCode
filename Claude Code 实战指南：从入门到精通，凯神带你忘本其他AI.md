## Claude Code 到底是什么？

Claude Code 是 Anthropic 推出的 AI 编程助手，直接跑在终端里。

但它和 Copilot、Cursor 这些不是一个玩意儿。它不是补全插件，它是一个真正能干活的"AI 员工"：

- 🧠 **理解自然语言**：说人话就行，不用记任何奇奇怪怪的命令
- ⚡ **主动执行操作**：读文件、写代码、跑命令、改 Bug，它来干
- 📂 **全项目上下文**：扫描整个代码库，给出符合你项目规范的建议
- 🖼️ **多模态理解**：粘贴截图它能分析，贴 UI 稿它能写代码

凯神最常说的一句话：**做好指挥官，让 AI 为你工作。**

| 对比维度 | 传统 AI 工具 | Claude Code |
|---------|------------|-------------|
| 交互方式 | 补全 / 对话框 | 终端自然语言 |
| 执行能力 | 仅提建议 | 直接执行操作 |
| 上下文理解 | 单文件 / 片段 | 全项目扫描 |
| 工作流集成 | 依赖 IDE 插件 | 原生终端集成 |
| 自定义深度 | 有限 | CLAUDE.md 深度定制 |
| 自动化能力 | 几乎没有 | Hooks 全流程自动化 |
